# 📖 企业微信智能机器人配置指南

## 前提条件

- 你所在的企业已开通企业微信
- 你有企业微信管理后台的访问权限（或者找管理员帮忙）

## Step 1：创建智能机器人

1. 登录 [企业微信管理后台](https://work.weixin.qq.com/wework_admin/frame)
2. 进入 **应用管理** → **应用** → 页面底部找到 **智能机器人**（或通过"管理工具"入口）
3. 点击 **创建智能机器人**
4. 填写信息：
   - **机器人名称**：热梗记录员（或你喜欢的名字）
   - **机器人描述**：团队热梗自动收集与周报生成
   - **Bot 模式**：选择 **API 模式**

## Step 2：配置回调 URL

创建完成后，进入机器人设置：

1. 在 **API 配置** 中：
   - **URL**：填入你的回调服务地址，格式为 `http://你的服务器IP:5000/wecom/callback`
   - **Token**：企微自动生成（也可以自定义），**复制保存**
   - **EncodingAESKey**：企微自动生成，**复制保存**

2. 记下以下信息：
   - **企业 ID (CorpID)**：在 "我的企业" → "企业信息" 页面找到
   - **Bot Token**：上一步复制的
   - **EncodingAESKey**：上一步复制的
   - **BotID**：机器人详情页显示

## Step 3：配置项目

将上述信息填入项目配置，有两种方式：

### 方式一：环境变量（推荐）

```bash
export WECOM_CORP_ID="你的企业ID"
export WECOM_BOT_TOKEN="你的Token"
export WECOM_BOT_AES_KEY="你的EncodingAESKey"
export WECOM_BOT_ID="你的BotID"
```

### 方式二：直接修改 config.py

编辑 `server/config.py`，填入对应的值。

## Step 4：启动回调服务

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python server/app.py
```

服务启动后，回到企微管理后台，点击 **验证URL**。如果配置正确，会显示验证成功。

## Step 5：把机器人拉进群

1. 在企业微信中创建一个群（或使用已有群）
2. 群设置 → 群机器人 → 添加 → 选择你刚才创建的智能机器人
3. 现在群成员就可以 @机器人 来录入热梗了

## Step 6：测试

在群里发消息测试：

```
@热梗记录员 城市不会眨眼 | 来自短视频的魔性台词 | https://v.douyin.com/xxx
```

如果机器人回复"✅ 已记录热梗：城市不会眨眼"，说明配置成功！

## 🌐 关于服务器

回调服务需要能被企微服务器访问到，有几种方案：

### 方案 A：公网服务器
直接部署在有公网 IP 的云服务器上。

### 方案 B：内网穿透
使用 ngrok、cpolar 等工具将本地端口暴露到公网：

```bash
# ngrok 示例
ngrok http 5000

# 然后将 ngrok 给的公网地址填到企微回调 URL 中
```

### 方案 C：公司内网
如果公司有统一的入口网关/反向代理，可以配置转发到你的服务。

## ⚠️ 注意事项

1. **HTTPS**：企微要求回调 URL 使用 HTTPS（开发阶段可先用 HTTP 测试）
2. **超时**：企微回调有 5 秒超时限制，处理逻辑要快
3. **重试**：如果你的服务没有返回 200，企微会重试最多 3 次
4. **图片限制**：群聊中 @机器人 只能发文本，图片需要私聊发送
