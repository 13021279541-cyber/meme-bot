7 周报生成与推送

「📊 周报」Tab 是整个系统的核心输出环节。

7.1 操作流程

  1. 选择日期范围（默认：本周一 ~ 今天）
  2. 点击「👀 预览周报」
    • 自动过滤灰色梗
    • 按等级排序：红 > 橙 > 蓝
    • 生成 16:9 宽屏 HTML 周报（四列卡片布局）
    • 前端截图生成缩略图预览（点击可全屏查看）
  3. 点击「📥 下载 HTML 周报」（可选）— 下载完整 HTML 文件到本地
  4. 点击「📣 推送到企微群」

7.2 推送三步流程

① Markdown 清单 → ② 截图图片 → ③ 同步腾讯文档
步骤内容说明
① Markdown 清单文字版热梗列表含编号、链接、概括、官号跟进、作者激励，广播到所有群
② 截图图片服务端 Puppeteer 自动截图自动压缩到 2MB 以内，以图片消息发送
③ 同步腾讯文档自动写入智能表格后台静默执行

7.3 周报样式

  • 封面页：标题「热梗洞察周报」+ 词云展示本期热梗 + 日期范围 + 署名「by 用户社区组」
  • 内容页：每页 4 张卡片，竖版图片区域 520px 高，无图卡片用随机 emoji 占位
  • 🎮 王者相关卡片有游戏手柄 badge 标注
  • 卡片包含：截图、梗名称、一句话概括、来源链接、官号跟进、作者激励

11 日常操作指南

场景 A：在群里快速录入一条梗

  1. 在企微群发截图（可选）
  2. @机器人 录入 梗名称 链接
  3. 收到确认回复即成功

场景 B：在网页批量管理

  1. 打开 http://21.6.179.196:3000
  2. 点 🔐 登录管理员
  3. 在「已录入」列表中设置分类、等级、补充概括
  4. 灰色 = 淘汰，其他等级进入周报

  1. 切到「📊 周报」Tab

  3. 点「👀 预览周报」确认内容
  4. 点「📣 推送到企微群」
  5. 等待推送完成（文字 + 图片 + 腾讯文档同步）
